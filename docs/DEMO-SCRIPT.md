# Demo script

## Goal
Show the full trust loop in under 4 minutes.

## Steps
1. Open homepage and say the thesis: buy information first, commitment later.
2. Choose **Agent Mode** and show the agent identity card.
3. Create a `DIRECT_DISPATCH` task with a redacted public summary, private destination, and staged micro-pricing.
4. Run planner preview and call out that the result changes the task path.
5. Post the task and note that the principal can still stop after any verified increment.
6. Open the **Task Command Center** and point to the stage ladder, public/private split, spend boundary, and agent controls.
7. Open the runner path, start the Self verification session, and keep the destination hidden.
8. Accept only after verification succeeds and show reveal data.
9. Submit a scout proof with an encrypted image bundle.
10. Return to the command center, review the proof, and release the scout increment.
11. Trigger an agent decision and explain why it continues or stops.
12. End on `/evidence`, explorer links, and the downloadable `agent.json` / `agent_log.json` artifacts.

## If asked about live chain state
- show Foundry tests
- show the shared escrow ABI export and deployed address file
- toggle the optional live write path in the UI if a wallet is available
- show explorer links for escrow, delegation policy, and any captured tx hashes
- explain that the contract now covers repeated heartbeats, disputes, timeout auto-release, and expiry refunds while the backend still handles encrypted proof media and reveal-token privacy
