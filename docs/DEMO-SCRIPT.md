# Demo script

## Goal
Show the full trust loop in under 4 minutes.

## Steps
1. Open homepage and say the one-liner.
2. Open buyer flow and create a queue job.
3. Point to the bounded permission policy: spend cap, expiry, token, contract, job binding.
4. Show what stays private before acceptance.
5. Open runner jobs list on a narrow/mobile viewport.
6. Open a runner job and point to the verification badge.
7. Explain that acceptance is gated by the Self-compatible adapter.
8. Show proof-hash submission for arrival / heartbeat / completion.
9. Show the staged payout timeline.
10. End on the fact that the buyer paid only as proofs arrived.

## If asked about live chain state
- show Foundry tests
- show contract ABI/output files
- show where Celo contract deployment slots into env config

11. Show live explorer links for escrow and policy contracts on Celo Sepolia.
